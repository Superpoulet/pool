== Pool

== Beschreibung
Plugin, um die Pool-Filtration temperaturabhängig zu verwalten.
Das Plugin unterstützt auch die folgenden Funktionen :

-	Sandfilter.
-	Pumpe für Reinigungsroboter.
-	aktives Überwintern.
- Solarheizungs Regelung.
