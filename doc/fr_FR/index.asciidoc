== Piscine

== Description
Plugin permettant de gérer la filtration d'une piscine en fonction de la température.
Le plugin gère également les fonctions suivantes :

-	Filtre à sable.
-	Surpresseur pour robot nettoyeur.
-	Hivernage actif.
-   Asservissement chauffage solaire.
